# Screenshots

Place screenshot images in this directory:

- `dashboard-screenshot.png` - Dashboard view with courses and stats
- `quiz-screenshot.png` - Quiz-taking interface
- `results-screenshot.png` - Results page with breakdown

## How to Create Screenshots

1. Open `index.html` in your browser
2. Navigate to each view
3. Take screenshots (1280x720 or 1920x1080 recommended)
4. Optimize images (use tools like TinyPNG or ImageOptim)
5. Save with the filenames above

## Tips for Good Screenshots

- Use a clean, populated state (not empty)
- Capture both desktop and mobile views
- Show actual data, not placeholders
- Use light or dark theme consistently
- Crop to show relevant UI only
